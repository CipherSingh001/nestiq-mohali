# 🏡 NestIQ — AI-Powered Real Estate Intelligence

> Find your perfect home in Mohali using plain English — no filters needed.

## 🚀 Live App
[**Open NestIQ →**](https://nestiq-mohali.streamlit.app)

## 🧠 What It Does
- **AI Property Search** — describe what you want in plain English, get ranked results
- **Investment Advisor** — ask any question about Mohali real estate, get data-backed answers  
- **Market Insights** — sector-wise price trends, appreciation rates, and comparisons

## ⚙️ Tech Stack
- **Frontend** — Streamlit
- **ML Model** — XGBoost (price prediction + property scoring)
- **AI/LLM** — Groq API (Llama 3.1) for natural language understanding
- **RAG System** — Sentence Transformers + FAISS for investment Q&A
- **Data** — 540+ real properties across 18 Mohali sectors
